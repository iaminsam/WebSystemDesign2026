📚 강의 개요

웹 시스템 설계 강의는 HTML, CSS, JavaScript를 기반으로 웹 애플리케이션의 구조와 동작 원리를 이해하고 웹 시스템을 설계하고 구현하는 능력을 기르는 것을 목표로 한다. 또한 Antigravity, Cursor, Codex를 포함한 AI 도구를 보조 도구로 활용하여 요구사항 분석, 설계 문서 작성, 코드 구현 및 코드 분석 능력을 함께 학습한다.

🎯 학습 목표

  • 웹의 기본 구조(DOM, 이벤트, HTTP)를 이해한다.
  • 웹 시스템의 구성 요소(클라이언트, 서버, 데이터 흐름)를 설명할 수 있다.
  • 웹 시스템 개발을 위한 요구사항을 분석하고 기능 명세로 정리할 수 있다.
  • 화면 흐름도와 기능 구조도를 기반으로 웹 시스템을 설계할 수 있다.
  • AI 도구를 활용하여 웹 시스템을 구현하고 생성된 코드를 분석·수정할 수 있다.
  • 웹 프로젝트 결과를 문서와 발표로 설명할 수 있다.

🧩 주요 학습 내용

  • HTML 구조와 웹 문서 이해
  • CSS 스타일링 및 레이아웃 설계
  • JavaScript 기초 문법
  • DOM 조작 및 이벤트 처리
  • 웹 시스템 설계 방법
  • AI를 활용한 코드 생성 및 분석
